# Nexura Store

[Live Demo](https://nexurastore.vercel.app)

Nexura is a full-featured e-commerce platform with a modern admin dashboard. It allows users to browse products, add items to the cart, and complete purchases, while admins can manage products, orders, and users.

## Features

### User-Facing

- Browse and search products
- Add/remove items from cart
- Checkout process

### Admin Dashboard

- Add, edit, and remove products
- Manage orders and view sales
- Manage user accounts

### Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (via Prisma ORM)
- **Deployment:** Vercel (frontend), Render/other backend hosting
